JDBC中常用的类和接口有以下：

  • DriverManager类：用于获取Connection
  • Connection接口：代表着对数据库的一次物理连接
  • Statement接口：代表着SQL语句的执行对象
  • PreparedStatement接口：代表着SQL语句的执行对象（预编译）
  • ResultSet接口：代表着查询的结果集
  • ResultSetMetaData接口：代表着元数据对象，可以通过元数据对象获取表名、字段名等信息
