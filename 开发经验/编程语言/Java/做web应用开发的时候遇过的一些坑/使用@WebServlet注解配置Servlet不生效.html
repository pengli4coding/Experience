问题描述：

为了方便，用@WebServlet注解来配置Servlet，眼看@WebServlet注解没有配置错耶，但是Servlet怎么也加载上，试了好几次都不行




解决方案：

问了一下度娘，原来问题出现在web.xml文件的上，在<web-app>标签中配置了metadata-complete属性为true，该metadata-complete属性用来指定当前的部署描述文件（web.xml）是否是完全的，如果设置为true的时候，则容器在部署的时候只会依赖当前的描述文件（web.xml），不会再去扫描注解来进行组件的部署。

因此，这个问题的解决方案是把web.xml文件中的<web-app>标签的metadata-complete属性为去掉，或者将该属性设置为false
