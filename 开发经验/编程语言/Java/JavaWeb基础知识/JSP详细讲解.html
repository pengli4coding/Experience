3 JSP的九个内置对象中的四个域对象

域对象的共同特点都是管理域中的属性（attribute）

它们都有着如下操作属性（attribute）的方法

  • void setAttribute(String name, Object o)
  • Object getAttribute(String name)
  • void removeAttribute(String name)

3.1 pageContext（范围：当前的页面内有效）

3.2 request（范围：当前的请求内有效）

3.3 session（范围：当前的会话内有效）

注意：会话是指浏览器不关（页面哪怕关闭），都是在一个会话范围内。如果浏览器关闭，那么会话就结束了

3.4 application（范围：web项目的运行生命周期内有效）

注意：web项目的运行生命周期是指服务器不重启或者项目不下线（浏览器或者页面哪怕都关闭了），都是在web项目的运行生命周期内。如果服务器重启或者项目下线，那么web项目的运行生命周期就结束了

4 JSP指令

JSP指令说白了就是指示应该如何生成JSP的真身Servlet，指令在“真身Servlet”中并不存在，而是生成“真身Servlet”需要用到指令

JSP指令格式：<%@指令名 attr1="" attr2=""%>

如：<%@ page language="java" contentType="text/html; charset=UTF-8" pageEncoding="UTF-8"%>

4.1 page指令

page指令常用的属性如下：

属性名 功能描述 常用设置
pageEncoding 设置当前JSP页面编码 pageEncoding="UTF-8"
contentType 设置往页面上写内容的格式，与在Servlet中设置response.setContentType("text/html; charset=UTF-8")是一样的 contentType="text/html; charset=UTF-8"
import 引包 import="java.util.*"
errorPage 指定如果当前页面发生错误的时候跳转到哪个错误页面（注意这里只能够设置当前的页面，不能够设置其它页面，而且这里的跳转是服务器转发，不用加项目名） errorPage="error.html"
idErrorPage 标识当前页面是否是错误页面，如果当前页面是错误页面，则可以使用JSP的内置对象exception isErrorPage="true"
isELIgnored EL表达式是否被忽略，默认为false，如果设置为true的话，EL表达式将不会生效 isELIgnored="false"

4.2 include指令

include指令用于做页面的静态包含，就相当于把两个页面静态拼凑在一起了

4.3 taglib指令

taglib指令用于在第三方标签库进行引入的时候，进行“导包”
