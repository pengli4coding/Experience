1 Filter介绍

1.1 什么是Filter？

Filter是JavaWeb三大组件（Servlet、Filter、Listener）之一，主要的作用是用于拦截客户端（浏览器）发过来的请求，使用Filter（拦截器）主要体现了面向切面编程的思想

1.2 Filter的常见应用场景

  • 客户端发请求过来，先判断用户的登录状态
  • 获取前台传参的中文乱码处理
