第一种方式：继承java.lang.Thread

具体步骤：

第一步：创建一个类继承java.lang.Thread

第二步：重写父类的run方法

第三步：创建类的实例

第四步：调用start方法开启线程

样例如下：

package com.pl.thread;

public class MyThread extends Thread{
	public static void main(String[] args) {
		MyThread t=new MyThread();
		t.start();//注意：这里调用的是start方法，而不是run方法
		for(int i=0;i<50;i++) {
			System.out.println(Thread.currentThread().getName()+i);
		}
	}

	@Override
	public void run() {//重写run方法
		for(int i=0;i<50;i++) {
			System.out.println(Thread.currentThread().getName()+i);
		}
	}
}

第二种方式：实现java.lang.Runnable接口

具体步骤：

第一步：创建一个类实现java.lang.Runnable接口

第二步：实现接口的run方法

第三步：创建类的实例

第四步：创建一个java.lang.Thread类的实例（传入上一步类的实例作为构造方法的参数）

第五步：调用start方法开启线程

样例如下：

package com.pl.thread;

public class MyThread implements Runnable{
	public static void main(String[] args) {
		MyThread my=new MyThread();//创建自定义类的实例
		Thread t=new Thread(my);//创建Thread类（构造方法中传入自定义类的实例作为参数）
		t.start();//注意：这里调用的是start方法，而不是run方法
		for(int i=0;i<50;i++) {
			System.out.println(Thread.currentThread().getName()+i);
		}
	}
	@Override
	public void run() {//重写run方法
		for(int i=0;i<50;i++) {
			System.out.println(Thread.currentThread().getName()+i);
		}
	}
}
