python整个开发生态的繁荣，很大程度依赖于成千上万的第三方库。可是在安装第三方库的时候，常常会遇到一些问题（报看不懂的错，没有依赖库等等，挺麻烦的），这些问题对于初学者来说会特别打击学习的兴趣。因此开这个帖子，用来总结常用的python第三方库的安装方法

方法一：pip install

最常用，但是由于下载的是国外的文件，有一定的几率下载失败。而且是下载文件回来编译，编译的过程中会出现一些问题

使用方法：pip install 库名

示例：pip install pandas

方法二：下载whl包进行安装

下载whl包，推荐一个下载的网站：美国的一所大学的网站

在cmd中进入上一步whl包的下载路径，执行pip install xxx.whl，注意：下载路径最好不要有中文
