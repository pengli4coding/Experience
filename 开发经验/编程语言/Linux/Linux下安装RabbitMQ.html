环境安装

第一步：安装预环境

yum -y install make gcc gcc-c++ kernel-devel m4 ncurses-devel openssl-devel

第二步：下载erlang的源代码文件并解压缩

下载网址：http://www.erlang.org/downloads

网盘地址：https://pan.baidu.com/s/1mj9m7XBVX3MyCUzZLDp4_Q   密码：8ave

解压安装包：tar zxvf otp_src_19.3.tar.gz

第三步：编译源码并安装erlang

分别运行以下几条命令

./configure --prefix=/usr/software/erlang

make

make install

第四步：配置erlang环境变量并重启服务器

在/etc/profile文件下追加以下内容

export ERLANG_HOME=/usr/software/erlang
export PATH=$PATH:$ERLANG_HOME/bin

第五步：测试erlang是否安装成功

输入erl命令，如果没有出现类似命名找不到的提示的话一般情况下就意味着安装erlang成功了

第六步：运行以下命令安装rabbitmq，并重启服务器

rpm --import https://www.rabbitmq.com/rabbitmq-release-signing-key.asc
# this example assumes the CentOS 7 version of the package
yum install rabbitmq-server-3.7.5-1.el7.noarch.rpm

配置rabbitmq的环境变量：在/etc/profile文件下追加以下内容

   
export PATH=$PATH:/usr/lib/rabbitmq/bin
启动rabbitmq

第一步：将rabbitmq设置为开机启动

输入命令：chkconfig rabbitmq-server on

第二步：启动rabbitmq

输入命令：/sbin/service rabbitmq-server start
