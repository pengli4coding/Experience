Set类型简介

Set集合是string类型的无序集合，set是通过hashtable实现的，对集合我们可以取交集、并集、差集




操作详解

相关命令：

命令 描述 示例 示例说明
SADD set member1 member2 member3 ... 向集合set中插入一个或多个成员值 sadd set1 aaa bbb ccc 向集合set1中插入aaa、bbb、ccc
SMEMBERS set 获取集合set中所有成员值 smembers set1 获取集合set1的所有成员值
SREM set member1 member2 member3 ... 删除集合set中一个或多个成员值 srem set1 aaa bbb 删除集合set1的成员值aaa和bbb
SPOP set 随机弹出集合set中的一个成员 spop set1 随机弹出集合set1中的一个成员
SINTER set1 set2 返回集合set1和集合set2的交集 sinter set1 set2 返回集合set1和集合set2的交集
SUNION set1 set2 返回集合set1和集合set2的并集 sinter set1 set2 返回集合set1和集合set2的并集
SDIFF set1 set2 返回集合set1和集合set2的差集，即set1中有的set2中没有的 sdiff set1 set2 返回集合set1和集合set2的差集，即set1中有的set2中没有的
SINTERSTORE interset set1 set2 返回集合set1和集合set2的交集存到interset中 sinter set1 set2 返回集合set1和集合set2的交集存到interset中
SUNIONSTORE unionset set1 set2 返回集合set1和集合set2的并集存到unionset中 sinter set1 set2 返回集合set1和集合set2的并集存到unionset中
SDIFFSTORE diffset set1 set2 返回集合set1和集合set2的差集存到diffset中 sdiff set1 set2 返回集合set1和集合set2的差集存到diffset中
SMOVE set1 set2 aaa 将集合set1中的成员aaa移动到set2中 smove set1 set2 aaa 将集合set1中的成员aaa移动到set2中
SCARD set 查看集合set中的成员个数 scard set1 查看集合set1中的成员个数
SISMEMBER set member 查看集合set中是否存在成员member sismember set1 aaa 查看集合set1中是否存在成员aaa
