什么是官方的employees示例数据库

官方的employees示例数据库是mysql官方给提供的示例数据库，我们的初学者可以用来进行数据库的操作实验，地址是https://launchpad.net/test-db




如何安装

第一步：下载

[root@VM_0_13_centos ~]# cd /usr/local/software
[root@VM_0_13_centos software]# wget https://launchpad.net/test-db/employees-db-1/1.0.6/+download/employees_db-full-1.0.6.tar.bz2


第二步：解压

[root@VM_0_13_centos software]# tar jxf employees_db-full-1.0.6.tar.bz2


第三步：导入数据

[root@VM_0_13_centos software]# cd employees_db
[root@VM_0_13_centos employees_db]# mysql -uroot -p123456 -t < employees.sql
