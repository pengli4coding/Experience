String类型简介

String类型是包含很多种类型的特殊类型，并且是二进制安全。比如序列化的对象进行存储，比如一张图片进行二进制存储，比如一个简单的字符串，数值等等。

String类型是redis中最基础的数据类型，其它的数据类型都是在String类型的基础上进行扩展的。




操作详解

相关命令：

命令 描述 示例
SET key value 设置指定key的值，如果指定key的值已经存在，则会覆盖原有的值 set name zhangsan
GET key 获取指定key的值 get name
GETSET key value 将给定key的值设为value，并返回key的旧值(old value) getset name lisi
SETNX key value 只有在key不存在时设置key的值，否则当key存在的时候设置key的值失败 setnx pet fatcat
SET key value XX 只有在key存在时设置key的值，否则当key不存在的时候设置key的值失败 set pet fatcat xx
MSET key1 value1 ...... keyN valueN 同时设置一个或多个 key-value 对 mset name xiaolili age 28
MGET key1 key2 ...... keyN 获取所有(一个或多个)给定 key 的值 mget name age
SETEX key seconds value 将值value关联到key，并将key的过期时间设为seconds(以秒为单位) setex name 10 xiaolili
PSETEX key milliseconds value 这个命令和 SETEX 命令相似，但它以毫秒为单位设置 key 的生存时间，而不是像 SETEX 命令那样，以秒为单位 psetex name 10000 xiaolili
INCR key 对key的值做++操作，并返回新的值。注意incr一个不是int的value会返回错误，incr一个不存在的key，则设置key值为1 incr age
INCRBY key increment 将key所储存的值加上给定的增量值（increment），key不存在时候会设置key，并认为原来的value是0 incrby age 5
DECR key 对key的值做--操作，并返回新的值。注意decr一个不是int的value会返回错误，decr一个不存在的key，则设置key值为-1 decr age
DECRBY key decrement 将key所储存的值减去给定的减量值（decrement），key不存在时候会设置key，并认为原来的value是0 decrby age 5
SETRANGE key offset value 用value参数覆写给定key所储存的字符串值，从偏移量offset开始 setrange email 10 #
GETRANGE key start end 返回key中字符串值的子字符 getrange email 10 11
STRLEN key 返回 key 所储存的字符串值的长度 strlen name
APPEND key value 如果 key 已经存在并且是一个字符串， APPEND 命令将指定的 value 追加到该 key 原来值（value）的末尾 append name world
