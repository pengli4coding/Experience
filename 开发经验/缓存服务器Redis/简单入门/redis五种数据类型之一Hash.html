Hash类型简介

Hash类型是String类型的field和value的映射表，或者说一个String集合。它特别适合存储对象，相比较而言，将一个对象类型存储在Hash类型里要比存储在String类型里面占用更少的内存空间，并方便存取整个对象。




操作详解

相关命令：

命令 描述 示例 示例说明
HSET key field value 设置指定哈希表key中字段field的值，如果指定哈希表key中字段field的值已经存在，则会覆盖原有的值 hset key1 name zhangsan 设置哈希表key1中字段name的值为zhangsan
HGET key field 获取指定哈希表key中字段field的值 hget key1 name 获取哈希表key1中字段name的值
HDEL key field 删除指定哈希表key的字段field hdel key1 addr 删除哈希表key1中的字段addr
HMSET key field1 value1 ...... fieldN valueN 设置哈希表key中多个字段的值 hmset key1 name zhangsan age 18 设置哈希表key1中字段name的值为zhangsan，字段age的值为18
HMGET key field1 ...... fieldN 获取哈希表key中多个字段的值 hmget key1 name age 获取哈希表key1中字段name和字段age的值
HKEYS key 获取哈希表key中的所有字段名 hkeys key1 获取哈希表key1中的所有字段名
HVALS key 获取哈希表key中的所有字段值 hvals key1 获取哈希表key1中的所有字段值
HGETALL key 获取哈希表key中的所有字段名以及对应的字段值 hgetall key1 获取哈希表key1中的所有字段名以及对应的字段值
HLEN key 获取哈希表key中的字段个数 hlen key1 获取哈希表key1中的字段个数
HSETNX key field value 只有在哈希表key的字段field不存在时设置值，否则当哈希表key的字段field存在的时候设置值失败 hsetnx key1 name lisi 设置哈希表key1中字段name的值为lisi，如果字段name已存在，则失败
HINCRBY key field increment 将哈希表key的字段field所储存的值加上给定的增量值（increment） hincrby key1 age 5 给哈希表key1中字段age的值加5，并返回加运算后的结果
