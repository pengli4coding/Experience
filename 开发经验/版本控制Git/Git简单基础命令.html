git init

把当前目录初始化为一个git仓库

这个命令通常不跟参数，进入一个文件夹之后在空白的地方右键然后点击Git Bash Here，在弹出的命令行窗口直接敲git init命令就可以使得当前目录变为git仓库了




git add 文件名

将文件变更信息提交到暂存区中




git rm 文件名

将工作区的文件删除，并将这次删除文件的变更信息提交到暂存区




git mv 旧文件名 新文件名

将工作区的文件重命名，并将这次重命名的变更信息提交到暂存区




git status

查看仓库的状态，如新增文件的信息未提交到暂存区，暂存区的信息未同步到本地仓库等




git commit -m 版本信息

提交暂存区的文件变更到本地仓库，并指定版本信息




git commit -am 版本信息

直接从工作区将文件的变更提交到到本地仓库，并指定版本信息




git log

查看当前分支的所有版本信息




git show 某一次提交的哈希值

查看某一次提交的文件内容的变更
