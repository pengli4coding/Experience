显示Git配置信息

查看用户级别配置信息的时候使用命令（先切换到用户目录~下）：git config --list

查看项目级别配置信息的时候使用命令（先切换到项目的目录下）：git config --list




设置提交仓库时的用户名信息

设置用户级别的时候使用命令：git config --global user.name '你的用户名'

设置项目级别的时候使用命令（要先切换到项目目录下）：git config --global user.name '你的用户名'




设置提交仓库时的邮箱信息

设置用户级别的时候使用命令：git config --global user.email '你的邮箱'

设置项目级别的时候使用命令（要先切换到项目目录下）：git config user.email '你的邮箱'




用户级别的配置文件和项目级别的配置文件

用户级别的配置文件位置：~/.gitconfig

若使用 git config 时用 --global 选项，读写的就是这个用户级别的配置文件



项目级别的配置文件位置：项目位置/.git/config

这里的配置仅仅针对当前项目有效，每一个级别的配置都会覆盖上层的相同配置，所以项目级别的配置文件会覆盖用户级别的配置文件中的同名变量
