什么是XML?

  • XML 指可扩展标记语言（EXtensible Markup Language）。
  • XML 是一种很像HTML的标记语言。
  • XML 的设计宗旨是传输数据，而不是显示数据。
  • XML 标签没有被预定义。您需要自行定义标签。
  • XML 被设计为具有自我描述性。
  • XML 是 W3C 的推荐标准。



XML和HTML之间的差异

项 XML HTML
主要应用领域 用来传输和存储数据 用来显示数据
设计焦点 数据的内容 数据的外观
标签是否固定 标签是固定的，不可以自定义标签 标签不是固定的，可以自定义标签
是否区分大小写 区分大小写 不区分



XML和properties（属性文件）比较

属性文件只能存储平面信息，而XML可以存储结构化信息。

解析属性文件只需要使用Properties类就可以了，而解析XML文档是很复杂的。




XML的版本

W3C在1998年2月发布 1.0版本

W3C在2004年2月发布1.1版本，但因为1.1版本不能向下兼容1.0版本，所以1.1没有人用。同时，在2004年2月W3C又发布了1.0版本的第三版。我们要学习的还是 1.0版本！！！




XML的主要应用场景

  • 程序的配置文件（这也是使用XML最常见的目的）。
  • 用于数据交换，不同语言之间用来交换数据（如webservice）。
  • 小型数据库，用来当数据库存储数据。
